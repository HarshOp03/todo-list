I Built a Task Management app . Using React and localstorage , deployed on vercel 


🚀 Features
Create & Manage: Easily add new tasks and delete completed ones.
Task Persistence: Saves your tasks so they remain even after refreshing the page.



📦 Installation & Setup on locally:

Bash
cd todo-app
Install dependencies:

Bash
npm install
Start the development server:

Bash
npm start
